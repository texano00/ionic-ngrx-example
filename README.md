<p>Features used:</p>
<ul>
  <li>@ngrx/store - RxJS powered state management for Angular applications, inspired by Redux</li>
  <li>@ngrx/effects - Side Effect model for @ngrx/store to model event sources as actions</li>
  <li>@ngrx/store-devtools - Store instrumentation that enables a powerful time-travelling debugger</li>
</ul>
